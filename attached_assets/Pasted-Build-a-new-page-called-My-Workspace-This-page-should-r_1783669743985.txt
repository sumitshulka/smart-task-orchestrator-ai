Build a new page called "My Workspace". This page should replace the traditional dashboard and become the user's personal work area after login.

Design Philosophy:

This is NOT a reporting dashboard.
This is the user's operational workspace.

The page should help the user start work immediately without navigating into Projects, Tasks or Defects.

Use a clean modern enterprise UI with cards, soft shadows, rounded corners, and generous spacing. The page should feel like Linear, Notion or ClickUp rather than Jira.

--------------------------------------------------------
SECTION 1 : AI Daily Brief
--------------------------------------------------------

Display a large AI briefing card at the top.

Example:

Good Morning Rahul 👋

Today you have:

• 7 active tasks
• 2 tasks due today
• 1 overdue task
• 3 unread workspace mentions
• 2 defects assigned
• 1 project needs your attention

Today's Suggestions

• Complete API Integration before 2 PM
• Review Defect #1045
• Respond to Rahul's mention in Student Portal

Provide buttons:

View My Work
Start Focus Mode
Refresh AI Summary

This AI summary should be generated from user assignments and notifications. Initially use rule-based logic. Future AI integration will replace this.

General Requirements

Do not use charts unless absolutely necessary.

Avoid clutter.

Everything should support one-click actions.

The page should feel like a personal productivity workspace rather than a management dashboard.

The design must be responsive.

Use reusable components.

Prepare the architecture for future AI integration.

Future AI features include:

Daily Brief

Task Prioritization

Workload Prediction

Smart Recommendations

Natural Language Commands

Build using the application's existing theme and component library.
