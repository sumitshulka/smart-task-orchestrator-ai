Build a feature called "Universal Search" for Tazq.

Purpose

Universal Search should become the primary navigation and discovery mechanism across the application.

It is not just a search box.

It should function like Spotlight, Raycast or Linear Command Menu.

Users should be able to find anything, navigate anywhere, and perform quick actions without browsing menus.

-------------------------------------------------------
Activation
-------------------------------------------------------

• Keyboard Shortcut: Ctrl + K (Cmd + K on Mac)
• Search icon in top navigation
• Opens as a centered modal overlay

-------------------------------------------------------
Search Scope
-------------------------------------------------------

Search across:

• Tasks
• Projects
• Milestones
• Defects
• Discussions
• Decisions
• Files
• Users
• Teams
• Clients (future)
• Reports
• Custom Fields
• Settings
• Workspace Messages

The architecture should allow additional entities to be added easily.

-------------------------------------------------------
Architecture
-------------------------------------------------------

Create a generic Search Provider architecture.

Each module should implement a SearchProvider.

Example:

TaskSearchProvider

ProjectSearchProvider

DefectSearchProvider

WorkspaceSearchProvider

Universal Search should aggregate results from all providers.

Future modules should only implement SearchProvider to become searchable.
