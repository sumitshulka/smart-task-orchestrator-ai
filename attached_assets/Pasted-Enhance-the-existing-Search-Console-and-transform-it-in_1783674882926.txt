Enhance the existing Search Console and transform it into a modern enterprise Command Center without changing its identity.

The Search Console should no longer function as a simple search box. It should become the central place from where users can search, navigate, create records, execute quick actions and eventually interact with AI.

Keep the existing search functionality but significantly enhance the user experience.

=========================================
Keyboard Experience
=========================================

Ctrl + K opens Search Console.

Arrow Keys navigate.

Enter executes.

Esc closes.

Tab switches sections.

Everything should be usable without touching the mouse.

=========================================
Design Philosophy
=========================================

The Search Console should become the fastest way to interact with Tazq.

Users should be able to find anything, open anything, create anything and perform common actions without navigating through menus.

The interface should feel modern, minimal, keyboard-first and inspired by products such as Linear, Raycast and Notion, while remaining consistent with Tazq's existing design language.
